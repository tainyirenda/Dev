Introduction

Welcome to the Afro Ubuntu TradeNet ecosystem. Our platform connects informal retailers with global suppliers, NGOs, and Fintech partners using AI-driven insights and WhatsApp-native automation.

WhatsApp Integration

Retailers interact with the TradeNet via WhatsApp. To perform an inventory check, simply send an image of your stock sheet or a list of items to our verified number.

Credit Scoring API

Fintech partners can access normalized credit tiers for merchants via the following endpoint:

GET /api/v1/analytics/credit-score?id={user_id}
